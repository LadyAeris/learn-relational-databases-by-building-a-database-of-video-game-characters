# Video Game Characters Database 🎮

This project was developed for the workshop **"Learn Relational Databases by Building a Database of Video Game Characters"**, part of the freeCodeCamp course **"Introduction to SQL and PostgreSQL"**.
The goal of this workshop was to practice SQL, PostgreSQL, database schema design and data manipulation inside a isolated environment.

## Tech Stack and Environment 🛠 
* **Database:** PostgreSQL
* **Containerization:** Docker Engine
* **OS Environment:** WSL 2 (Ubuntu)
* **IDE:** Visual Studio Code

## What I Practiced 📑
* Creating and managing relational database tables.
* Defining primary keys, foreign keys and column constraints.
* Writing advanced SQL queries to filter and join character data.

*Note: This repository contains the final code generated during the freeCodeCamp interactive environment.*
